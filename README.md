## Open Source

### [libreFS](https://github.com/libreFS) — founder & maintainer

A community fork of MinIO, the S3-compatible object storage server, created after MinIO Inc. moved LDAP/OIDC authentication, the WebUI, distributed mode, and pre-compiled binaries into their proprietary AIStor product. libreFS keeps all of that freely available under AGPL-3.0.

- Migrated 6 Go SDK libraries from `github.com/minio/*` to `github.com/libreFS/*`, preserving full commit history and cross-module dependency graph
- Designed and shipped CI/CD pipelines (lint, vulnerability scanning, multi-version matrix, GitHub Releases) for each library repo
- Fixed multi-arch Docker builds (amd64 + arm64) across server, CLI client, and web console
- Maintains full S3 API and admin API compatibility while cherry-picking upstream security and bug fixes
